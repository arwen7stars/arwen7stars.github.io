Lead Your Way Home!

Home Pricing is a free mobile application whose main objective is to help people finding properties in the real estate market and collect information about existing properties in the area chosen by the user. The main goal is to provide a list of houses that best matches the requirements of the users.

This project was proposed to my group by Deloitte, a widely renowned multinational enterprise in the context of Project Management Laboratory @ FEUP.

Area: Android Development

Languages & Tools: Javascript React Native Node.js Express.js MongoDB Scrapy Python Docker Git
